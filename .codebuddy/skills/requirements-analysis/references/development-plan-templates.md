# 开发计划模板

> **技术聚焦说明**: 本文档中的开发计划模板默认基于 **C++/Qt** 技术栈。

- **默认语言**: C++ (C++11)
- **默认 GUI**: Qt Widgets 或 Qt Quick/QML
- **默认构建**: qmake + CMake (混合使用)
- **默认测试**: Google Test / Qt Test
